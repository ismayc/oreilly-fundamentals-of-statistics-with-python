Common Pitfall: np.std() defaults to the population standard deviation (ddof=0), while pandas .std() defaults to the sample standard deviation (ddof=1). If you are treating your data as a sample, pass ddof=1 to np.std() so the two agree. Mixing the defaults is a common source of slightly different numbers.

Common Pitfall: Do not confuse the standard deviation of the raw data with the standard deviation of the sample means (the standard error). The histogram below shows the spread of sample means, which is much narrower than the spread of the original population. By the Central Limit Theorem, the sampling distribution of the mean becomes approximately normal as the sample size grows, even when the population itself is skewed.

Exercise 2.1: Advanced Plots

  • Datetime Accessor .dt:
    • The .dt accessor is used with pandas Series that contain datetime objects.
    • It provides access to datetime-specific attributes and methods.
  • Convert to a Period with Monthly Frequency:
    • The method .to_period('M') converts each datetime value into a Period object with a monthly frequency.
    • The argument 'M' stands for “month”. This means that each date is represented as a period corresponding to its month and year.
    • For example, if grading_date is 2023-03-15, converting it with .to_period('M') will result in a period like 2023-03.

Walkthrough 2.4: Correlations

Common Pitfall: Correlation measures only the strength of a linear association, and a strong correlation does not imply that one variable causes the other. Two variables can be highly correlated because of a third, lurking variable. Also, .corr() (Pearson) can miss strong non-linear relationships, so always pair it with a scatter plot.

Walkthrough 3.2: t-tests

Common Pitfall: A p-value is not the probability that the null hypothesis is true. It is the probability of seeing data at least as extreme as yours if the null hypothesis were true. A small p-value is evidence against the null, not a statement about how likely the null is.

Common Pitfall: The alternative argument controls whether the test is one-tailed ('greater' or 'less') or two-tailed ('two-sided', the default). Pick the direction before you see the data, based on the question. A one-tailed test only detects an effect in the specified direction, so using the wrong tail can hide a real effect.

Walkthrough 3.4: Non-Parametric Tests

Common Pitfall: Non-parametric tests like Mann-Whitney U and Kruskal-Wallis do not assume normally distributed data, which makes them useful for skewed data such as salaries. But they compare distributions (ranks/medians) rather than means, so do not report them as tests of the difference in means. Reach for them when the normality assumption of a t-test or ANOVA is clearly violated.
